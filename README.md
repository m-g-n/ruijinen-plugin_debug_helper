# 類人猿デバッグヘルパー
類人猿のプロダクトのデバッグをより楽にできるように補佐機能を実装しています

## 機能
- ページ表示時もページ上部のセレクトボックスで表示レイアウトを切り替え可能

## このプラグイン動作の前提条件
- Snow Monkeyテーマがインストールされてない環境では動きません
- 各機能は管理者権限のみ使えます
